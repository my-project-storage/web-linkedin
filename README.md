# 링크드인 클론

클론 코딩으로 따라하며 배웁니다!

## 사용 기술

#### Language

- `Typescript`
- `HTML`, `SCSS`

#### Backend

- `nestJS`
- `Postgres`

#### Frontend

- `Angular`
- `Ionic`

## 준비

- 프레임워크
  ```bash
  npm i -g @nestjs/cli
  npm i -g @ionic/cli
  ```
- 환경 변수
  ```
  # .env
  POSTGRES_HOST=
  POSTGRES_POST=
  POSTGRES_USER=
  POSTGRES_PASSWORD=
  POSTGRES_DATABASE=
  ```

## 실행

```bash
# backend develop
npm run start:dev

# frontend
ionic serve
```
